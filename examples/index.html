User Feedback Widget Examples

This page demonstrates different ways to integrate and use the feedback widget.

Star Rating Feedback


const feedbackWidget = new FeedbackWidget({
  type: 'star',
  targetElement: document.getElementById('star-container'),
  theme: {
    primaryColor: '#007BFF',
    secondaryColor: '#6C757D',
    fontFamily: 'Arial, sans-serif'
  },
  onSubmit: function(data) {
    console.log('Feedback submitted:', data);
  }
});
feedbackWidget.init();
